## 如何替换图片

### 最简单的方法：
1. 把你准备好的图片重命名为对应的文件名
2. 放进对应的文件夹
3. 刷新浏览器 — 搞定！

### 角色图片要求：
- 格式：PNG（推荐，支持透明背景）
- 尺寸：至少 400×600 像素
- 建议用 **全身立绘** 或 **半身像**
- 透明背景效果最好

### 如果没有图片：
- 游戏会自动显示彩色占位符（角色名字的圆圈）
- 不影响游戏运行！可以先玩起来，图片以后慢慢补

## 游戏内容扩展

游戏对话全部在 `index.html` 中的 JavaScript 里，搜索 `// === DAY 1 CONTENT ===` 即可找到。
你可以直接修改对话文字，不需要懂编程。
